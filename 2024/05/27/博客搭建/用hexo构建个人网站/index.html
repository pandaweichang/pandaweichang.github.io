构建个人网站

发表于 2024-05-27 更新于 2024-08-15 分类于 博客搭建

本文将通过github pages+hexo的方式，构建一个个人网站。

简介

Github Pages是什么

是GitHub官方提供的一种免费的静态站点托管服务，可以在Github仓库里托管和发布自己的静态网站页面

Hexo是什么

一个静态博客框架，基于Node.js运行

准备工作

安装git

下载并安装git

安装Node.js

下载并安装Node.js

安装Hexo博客架构

下载hexo

npm指令中 npm install -g 是指全局下载。一般下载在C:\Users\xxxx\AppData\Roaming

安装完成后输入

查看版本

初始化hexo

一般打开cmd是在C盘根目录下，初始化的文件夹位置会生成在根目录下

我这里选择把文件放在其他位置，在选择好的位置E盘下新建文件夹MYblog

进入对应文件夹

cd是进入对应文件夹操作

完成后，会在进入了的MYblog文件夹下生成初始化文件并存放在你命名的blog文件夹里

安装npm

进入blog文件中，安装npm

安装完成后会出现一些文件，部分文件作用如下

文件名 作用
_config.yml 网站配置文件
package.json 应用程序信息
scaffolds 模板文件夹(存放文章模板)
source 存放用户资源文件夹（所写文章存放）
themes 主题文件夹

启动服务站点

此时可以访问本地服务器进入网站http://localhost:4000/
